5.2 Lgistic Regression

\[logit(y) = \ln\left(\dfrac{y}{1-y}\right) = c_0 + c_1x_1 + c_2X_2 + \cdots + c_kx_k\] \[\dfrac{y}{1-y} = e^{c_0 + c_1x_1 + c_2X_2 + \cdots + c_kx_k}\] [y = \dfrac{1}{1+e{-(c_0 + c_1x_1 + c_2X_2 + \cdots + c_kx_k)}}]
